# TYAttributedLabel
TYAttributedLabel 简单易用的属性文本的控件，支持富文本，图文混排显示，支持添加链接，image和UIView控件，支持自定义排版显示

## 截图如下
![image](https://raw.githubusercontent.com/12207480/TYAttributedLabel/master/screenshot/TYAtrributedLabelDemo.gif)

## Requirements
* Xcode 5 or higher
* Apple LLVM compiler
* iOS 6.0 or higher
* ARC
